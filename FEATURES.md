## ğŸŒ Page Context

### When to Use
- Asking about specific webpage content
- Summarizing articles
- Extracting information from pages
- Comparing multiple pages

### How to Use
1. Click **"Page Context"** to expand panel
2. Check **"Current Page"** for active tab
3. Check any **"Other Tabs"** to include them
4. Badge shows number of selected pages
5. Ask your question

### Context Badge
- **0** - No context (general questions)
- **1** - One page selected
- **2+** - Multiple pages selected
