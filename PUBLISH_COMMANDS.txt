================================================================================
TROUBLESHOOTING
================================================================================

If you get "Permission denied (publickey)":
- Your SSH key is not set up with GitHub
- Solution: Use HTTPS instead:
  git remote set-url origin https://github.com/phaupaul/nova-ai-assistant.git

If you get "push rejected":
- Run: git pull origin main --rebase
- Then: git push

If you get authentication errors:
- Use Personal Access Token from: https://github.com/settings/tokens
- Or set up SSH key: https://docs.github.com/en/authentication
